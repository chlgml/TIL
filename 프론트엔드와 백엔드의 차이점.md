### 프론트엔드와 백엔드의 차이점

프론트엔드(front-end)

- 웹을 사용할 때 사이트에 직접 보이는 부분

   (사용자들이 웹사이트에서 보고 상호작용하는 모든 것들)

- 프론트 엔드 개발로는 HTML, CSS, Javascript의 언어를 사용한다.



백엔드(BACK-END)

- 웹을 사용할 때 사이트에 보이지 않는 부분

  (프론트엔드에 있는 사용자들이 취하는 행동을 처리함 서버 코딩, 서버상에서 DB활용, API제작등)

  -  API란?  Application program Interface의 약자로 프로그램들이 서로 상호작용하는 것을 도와주는 매게체로 볼 수 있다.

- 백엔드 개발로는 PHP, Node.js, Javascript, C++, Java, Python의 언어를 사용한다.



프론트엔드 VS 백엔드

- 프론트엔드는 보이는 부분을 말하지만 백엔드는 보이지 않는 부분을 말한다.  -->  결국 둘은 다루는 부분이 다르다.
- 백엔드랑 프론트엔드를 개발할 때 쓰는 언어는 다르다.
